# LoLinfo
App to display your ranked stats and last 3 games!
Made using python. 
## Setup
1.
```
pip install tkinter
pip install riotwatcher
```
<br />
2. Clone repository <br />
3. Get API key from https://developer.riotgames.com and put it into apikey.py <br />
4. Run!


## Libraries: 
tkinter
riotwatcher
